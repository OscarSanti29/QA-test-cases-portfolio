# QA-test-cases-portfolio

This repository contains sample manual QA test cases, bug reports, and test summaries created for practice and demonstration.

## Contents
- **test-cases/**: Manual test cases written for web apps (Markdown + CSV)
- **bug-reports/**: Bug report template + example bug report
- **test-summary/**: Example test summary report

## Projects Tested
- FakeStore (React + TypeScript + API)
- MealFinder (React + JavaScript + API + Local Storage)

## Testing Types Covered
- Smoke Testing
- Regression Testing
- Exploratory Testing
- UI Validation
- Basic API-driven UI checks

## Notes
These artifacts are created as portfolio examples to demonstrate QA documentation skills, clarity, and structured testing.
